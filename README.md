# CodSoft-Task1-Titanic
# CodSoft Data Science Internship

## Task 1: Titanic Survival Prediction

### Objective

The objective of this project is to build a Machine Learning model that predicts whether a passenger survived the Titanic disaster based on passenger information.

### Dataset

* Titanic Dataset
* Source: https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv

### Tools & Technologies

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

### Machine Learning Algorithm

* Random Forest Classifier

### Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Exploration
4. Data Preprocessing
5. Feature Selection
6. Train-Test Split
7. Model Training
8. Prediction
9. Model Evaluation
10. Data Visualization

### Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Classification Report

### Output

The model predicts passenger survival with good accuracy and visualizes important insights using graphs.
